### Getting Started  

> Pré-condition:  
>
> - yarn  
> - npm  

- clone this repository  
`git clone https://github.com/cezarcozta/mind-web.git`  

- access repository folder  
`cd mind-web`  

- install dependencies  
`yarn`  

- run the project  
`yarn start`  

- access locahost on web browser  
  - [localhost](https://localhost:3000)
